## [525. Contiguous Array](https://leetcode.com/problems/contiguous-array/)


Given a binary array `nums`, return _the maximum length of a contiguous subarray with an equal number of_ `0` _and_ `1`.

**Example 1:**

```
Input: nums = [0,1]
Output: 2
Explanation: [0, 1] is the longest contiguous subarray with an equal number of 0 and 1.
```

**Example 2:**

```
Input: nums = [0,1,0]
Output: 2
Explanation: [0, 1] (or [1, 0]) is a longest contiguous subarray with equal number of 0 and 1.
```

**Constraints:**

*   <code>1 <= nums.length <= 10<sup>5</sup></code>
*   `nums[i]` is either `0` or `1`.



## Solution

- Brute Force	$Time: O(n^2), Space: O(1)$ 	(Time Limit Exceeded)

```go
func findMaxLength(nums []int) int {
    n := len(nums)
    maxLen := 0
    for i := range nums {
        counter := 0
        for j := i; j < n; j++ {
            if nums[j] == 1 {
                counter++
            } else {
                counter--
            }
            if counter == 0 {
                maxLen = max(maxLen, j-i+1)
            }
        }
    }
    return maxLen
}
```



- PrefixSum + HashMap	$Time: O(n), Space: O(1)$ 

```go
func findMaxLength(nums []int) int {
    maxLen := 0
    // map: counter -> index
    m := map[int]int{0: -1}     // for empty subarray A[0..0)
    counter := 0
    for i, v := range nums {
        if v == 1 {
            counter++
        } else {
            counter--
        }
        // find previous location having same counter
        if pre, ok := m[counter]; ok {
            maxLen = max(maxLen, i-pre)
        } else {
            m[counter] = i
        }
    }
    return maxLen    
}
```
